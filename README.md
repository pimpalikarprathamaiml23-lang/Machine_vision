# Machine Vision Project using OpenCV (Google Colab)

## 📌 Overview

This project demonstrates basic machine vision operations using OpenCV in Google Colab.

## 📊 Features

* Image loading and display
* Image properties extraction
* Pixel value analysis
* RGB color channel separation
* Grayscale conversion
* Video loading and playback

## 🛠️ Technologies Used

* Python
* OpenCV
* NumPy
* Matplotlib
* Google Colab

## 📂 Project Files

* machine_vision.ipynb → Main notebook
* sample images/videos (optional)

## 🚀 How to Run

1. Open Google Colab
2. Upload the notebook
3. Run all cells step by step

## 📷 Output

* Image visualization
* Color channel separation
* Grayscale conversion
* Video frame processing
